# hana-marketing-company

Claude Code でマーケティング会社の仮想組織を構築・運営するプラグイン。

`/hana_marketing_company` を実行すると、CEOがあなた専用の窓口になります。3ステップで即運用開始。

## インストール

```
/plugin marketplace add [YOUR_GITHUB_USERNAME]/hana-marketing-company
/plugin install hana_marketing_company@hana-marketing-company
```

## コンセプト

```
あなた → CEO（窓口） → 各部署（12部署）
```

- **CEO**: 常に窓口。全ての指示を受けて最適な部署に振り分け
- **秘書室**: TODO管理、壁打ち、メモ、何でも相談
- **マーケ統括**: コンテンツ・SNS・イベント・メルマガ・PR・データ分析・ブランディング
- **その他**: CS部、総務部、組織開発部

## 組織構成

### CEO直下
| 部署 | 担当領域 |
|------|---------|
| 秘書室 | TODO管理、壁打ち、メモ、相談（常設） |
| 総務部 | 雑務、管理業務全般 |
| 組織開発部 | 研修・ワークショップ企画、メンバー育成、組織課題解決 |

### マーケティング統括
| 部署 | 担当領域 |
|------|---------|
| マーケ戦略部 | 市場調査、戦略立案、KPI管理 |
| コンテンツ制作部 | ブログ、SEO記事、ホワイトペーパー、お役立ち資料 |
| SNS運用部 | SNS投稿企画・運用・分析 |
| イベント企画部 | ユーザー会、セミナー、ウェビナー |
| メルマガ部 | メールマーケティング、配信企画 |
| PR・広報部 | プレスリリース、メディア対応 |
| データ分析部 | 施策のKPI追跡、キャンペーン分析、ROI測定 |
| ブランディング部 | ブランド設計・管理、CI、ガイドライン |

### その他
| 部署 | 担当領域 |
|------|---------|
| CS部 | カスタマーサクセス、顧客満足度向上 |

## 対応サービス

各部署内でサービス別にフォルダ管理:

- 勤怠管理システム
- 経費精算システム
- WEB明細配信システム
- HRコンサルティング
- BPOサービス

## 使い方

### 初回セットアップ（3ステップ）

```
> /hana_marketing_company

CEO: はじめまして！CEOとして、あなたの会社経営をサポートします。
     まず、現在の事業状況を教えてください。
あなた: 5つのHR系サービスのマーケティングを担当しています

CEO: 今の事業目標や、注力したいことは？
あなた: リード獲得を2倍にしたい

CEO: 今一番困っていることは？
あなた: コンテンツの制作が追いつかない

→ .hana_marketing_company/ が自動生成される（完了！）
```

### 日常の運営

```
> /hana_marketing_company
CEO: おはようございます！何かありますか？

> 今日やること教えて
CEO: 今日のTODOです:
  - [ ] 勤怠管理のSEO記事を入稿
  - [ ] ウェビナーの振り返りMTG

> 勤怠管理のブログ記事を企画して
CEO: 承知しました。コンテンツ制作部で企画を進めます。
  → content/kintai/ に企画ファイルを保存

> 来月のウェビナーを企画したい
CEO: いいですね！イベント企画部で進めましょう。
  → events/webinars/ に企画ファイルを作成
```

## PC移行時のバックアップ

プラグイン本体はGitHubから再インストール。データは `.hana_marketing_company/` フォルダをコピーするだけ。

```
# 新PCでの復元
1. Claude Code をインストール
2. プラグインをインストール（上記コマンド）
3. .hana_marketing_company/ フォルダを元の場所にコピー
4. /hana_marketing_company で運営再開！
```
