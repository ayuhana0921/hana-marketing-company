# イベント企画部 (Events Planning)

## 役割・責務

イベント企画部は外部向けのイベント・ユーザー会を企画・運営する部門です：
- セミナー・ウェビナーの企画・運営
- ユーザー会の企画・開催
- 参加者募集・管理
- 当日運営・事後フォロー
- イベントの成果測定

## フォルダ構成

```
events/
├── seminars/               # セミナー企画・実績
│   ├── kintai/             勤労の獅子
│   └── hr-consulting/      HRコンサルティング
├── webinars/               # ウェビナー企画・実績
│   ├── kintai/             勤労の獅子
│   └── hr-consulting/      HRコンサルティング
└── user-meetings/          # ユーザー会企画・実績
    ├── kintai/             勤労の獅子
    └── hr-consulting/      HRコンサルティング
```

## サービス別フォルダの使い方

- `seminars/kintai/` : 36協定・働き方改革セミナー企画
- `webinars/kintai/` : 勤怠管理ウェビナー企画
- `*/hr-consulting/` : HR制度・組織改善をテーマにしたイベント
- ファイル命名: `YYYYMMDD_イベント名.md`

## 基本ルール

1. **ファイル命名**
   - `seminars_イベント名_YYYYMMDD.md` - セミナー記録
   - `webinars_テーマ_YYYYMMDD.md` - ウェビナー記録
   - `user-meetings_開催地_YYYY.md` - ユーザー会記録

2. **企画・運営**
   - 2ヶ月前から企画スタート
   - 登壇者調整・資料準備を事前に完了
   - 参加者募集はマーケティング各部門と連携
   - 当日は進行管理・問題対応を実施

3. **事後フォロー**
   - 参加者アンケート・フィードバック収集
   - 次回改善点をまとめて記録
   - 顧客情報はCS部に共有

## イベント種別

- **セミナー**: 対面・ハイブリッド形式、各サービスの導入例紹介
- **ウェビナー**: オンライン形式、労務改革・経費管理のトレンド
- **ユーザー会**: 既存顧客交流、事例共有、ニーズヒアリング

## クロスファンクション連携

- **秘書室**: 重要イベント・経営陣出席イベントの日程調整
- **コンテンツ制作部**: セミナー資料・ホワイトペーパー制作依頼
- **SNS運用部**: イベント告知・参加促進投稿
- **メルマガ部**: 参加者募集メール配信
- **PR部**: プレスリリース・メディア対応
- **CS部**: 参加者のアフターフォロー・営業情報共有
