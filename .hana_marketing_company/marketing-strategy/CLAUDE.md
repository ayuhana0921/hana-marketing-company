# マーケ戦略部 (Marketing Strategy)

## 役割・責務

マーケ戦略部はマーケティングの方向性を策定し、全社の成長を牽引する部門です：
- 市場調査・競合分析
- マーケティング戦略の策定
- KPI設定・モニタリング
- 施策の効果測定・改善提案
- 各部門への目標指示

## フォルダ構成

```
marketing-strategy/
├── research/       # 市場調査・競合分析（サービス横断）
├── kpi/            # KPI管理・進捗管理（全サービス統合）
├── persona/        # ペルソナ・ターゲット定義（サービス別記述）
└── annual-plan/    # 年間・四半期マーケティング計画
```

## 戦略はサービス別ではなく横断管理

マーケ戦略部は市場全体を俯瞰する役割のため、サービスごとにフォルダを分けない。
代わりに各ファイルの**中でサービス別セクションを立てる**形で管理する。

例:  の中に以下を記述
- 勤労の獅子のターゲット（中堅・大企業の労務担当者）
- HRコンサルティングのターゲット（HR部門強化を目指す企業の人事責任者）

## 基本ルール

1. **ファイル命名**
   - `research_テーマ_YYYY.md` - 年単位の調査（例: `research_競合_2026.md`）
   - `kpi_YYYY-MM.md` - 月単位のKPI集計
   - `persona_YYYY.md` - 年次更新のペルソナ定義
   - `annual-plan_YYYY.md` - 年間計画・四半期ロードマップ

2. **調査・分析**
   - 定性・定量の両面からの分析を実施
   - 経営層への報告時期を事前に予定
   - 他部門の実績データを集約

3. **KPI管理**
   - 全社目標を月次で追跡
   - 未達成項目は改善施策と共に報告
   - 成功事例は他部門へ共有

## クロスファンクション連携

- **秘書室**: 四半期・年間目標の策定
- **データ分析部**: 詳細分析・ダッシュボード依頼
- **全マーケ部門**: 月次目標の設定・進捗確認
