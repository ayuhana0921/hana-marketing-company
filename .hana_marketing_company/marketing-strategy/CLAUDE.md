# マーケ戦略部 (Marketing Strategy)

## 役割・責務

マーケ戦略部はマーケティング全部門の**上位部門**として、方針・KPI・テーマを統括する部門です：
- 市場調査・競合分析・ペルソナ定義
- マーケティング戦略の策定と全部門への方針指示
- KPI設定・モニタリング・進捗管理
- 施策の効果測定・改善提案
- 年間・四半期計画の策定と各部門への落とし込み

## 管掌部門（マーケ戦略部の指揮下）

以下の部門はマーケ戦略部の方針に従って動く：

| 部門 | 主な連携内容 |
|------|------------|
| コンテンツ制作部 | コンテンツテーマ・SEOキーワード方針・制作優先度の指示 |
| SNS運用部 | 投稿テーマ・キャンペーン方針・月次目標の設定 |
| イベント企画部 | 開催テーマ・ターゲット・集客目標の設定 |
| メルマガ部 | 配信テーマ・セグメント戦略・目標開封率の設定 |
| PR・広報部 | PR施策テーマ・媒体戦略・メッセージングの承認 |
| データ分析部 | KPI定義・レポート要件・分析依頼 |
| ブランディング部 | ブランド方針・トンマナ・ビジュアル基準の策定 |

## フォルダ構成

```
marketing-strategy/
├── research/       # 市場調査・競合分析（サービス横断）
├── kpi/            # KPI管理・進捗管理（全サービス統合）
├── persona/        # ペルソナ・ターゲット定義（サービス別記述）
└── annual-plan/    # 年間・四半期マーケティング計画
```

## 戦略はサービス別ではなく横断管理

マーケ戦略部は市場全体を俯瞰する役割のため、サービスごとにフォルダを分けない。
代わりに各ファイルの**中でサービス別セクションを立てる**形で管理する。

例:  の中に以下を記述
- 勤労の獅子のターゲット（中堅・大企業の労務担当者）
- HRコンサルティングのターゲット（HR部門強化を目指す企業の人事責任者）

## 基本ルール

1. **ファイル命名**
   - `research_テーマ_YYYY.md` - 年単位の調査（例: `research_競合_2026.md`）
   - `kpi_YYYY-MM.md` - 月単位のKPI集計
   - `persona_YYYY.md` - 年次更新のペルソナ定義
   - `annual-plan_YYYY.md` - 年間計画・四半期ロードマップ

2. **調査・分析**
   - 定性・定量の両面からの分析を実施
   - 経営層への報告時期を事前に予定
   - 他部門の実績データを集約

3. **KPI管理**
   - 全社目標を月次で追跡
   - 未達成項目は改善施策と共に報告
   - 成功事例は他部門へ共有

## 意思決定フロー

```
ユーザー → CEO → マーケ戦略部（方針決定）
                       ↓
         コンテンツ／SNS／イベント／メルマガ／PR／分析／ブランディング
                       ↓
                  実行・報告
                       ↓
              マーケ戦略部（進捗確認・改善指示）
```

## クロスファンクション連携

- **CEO**: 全社方針・重要意思決定の確認・エスカレーション先
- **秘書室**: 四半期・年間目標の策定支援・スケジュール管理
- **データ分析部**: KPI詳細分析・ダッシュボード依頼（マーケ戦略部の直轄分析チーム的役割）
- **CS部**: 顧客インサイト・成功事例のフィードバック収集
