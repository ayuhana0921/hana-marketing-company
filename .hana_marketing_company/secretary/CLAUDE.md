# 秘書室 (Secretary Office)

## 役割・責務

秘書室は経営陣のサポート、情報の一元管理、全社調整を担う部門です。CEO直下に位置し、以下の業務を統括します：
- スケジュール管理・アポイント調整
- 重要な意思決定のサポート（壁打ち、メモ化）
- 全社への重要通知・連絡
- 情報資産の一元化と共有
- 部門間の連携調整

## フォルダ構成

```
secretary/
├── inbox/          # 受信・相談事項の管理
├── todos/          # CEO・経営陣の優先業務
├── notes/          # 重要議議事録・メモ
```

## 基本ルール

1. **ファイル命名**
   - `inbox_YYYYMMDD.md` - 日単位の受信管理
   - `todos_YYYY-MM.md` - 月単位のタスク管理
   - `notes_テーマ_YYYYMMDD.md` - テーマ別メモ

2. **情報の機密性・優先度付け**
   - 経営レベルの情報は厳格に管理
   - 各部門からの相談は対応順序を記録
   - 完了後は完了日時を記載

3. **部門間連携**
   - 全社に関わる重要事項は秘書室から各部門に通知
   - 部門の課題や調整が必要な場合は当事者部門に連絡
   - 月1回の全体調整会議の議事録を保管

## クロスファンクション連携

- **CEO**: 毎朝のスケジュール確認、重要意思決定サポート
- **全部門**: 重要通知・期限・重要事項の共有
- **組織開発部**: 研修企画・人事課題相談
- **マーケ戦略部**: 経営方針・四半期目標の策定
