# SNS運用部 (SNS Operations)

## 役割・責務

SNS運用部はソーシャルメディアの企画・運用・分析を統括する部門です：
- SNS投稿計画・実行
- エンゲージメント管理・コミュニティ運営
- SNS分析・レポート作成
- トレンド・キャンペーン企画
- フォロワー増加施策の推進

## フォルダ構成

```
sns/
├── posts/                  # 投稿計画・実績（サービス横断）
│   ├── kintai/             勤労の獅子
│   └── hr-consulting/      HRコンサルティング
└── analytics/              # 分析レポート
```

## サービス別フォルダの使い方

- `posts/kintai/` : 勤労の獅子に特化したSNS投稿案・キャンペーン施策
- `posts/hr-consulting/` : HRコンサルに特化したSNS投稿案・専門コンテンツ
- ファイル命名: `YYYYMM_テーマ.md`（例: `202604_kintai_campaign.md`）

## 基本ルール

1. **ファイル命名**
   - `posts_YYYY-MM_プラットフォーム.md` - 月単位の投稿計画
   - `analytics_YYYY-MM.md` - 月単位の分析レポート

2. **投稿管理**
   - 月次投稿計画で日程・内容を事前決定
   - 定時投稿＋ リアルタイム投稿の両方で運用
   - 投稿素材はコンテンツ制作部から活用
   - コメント対応・フォロワー対応を記録

3. **分析・改善**
   - 週次でエンゲージメント数値をチェック
   - 月次レポートでプラットフォーム別の成果を分析
   - 改善提案を月1回マーケ戦略部に報告

## 対象プラットフォーム

- **LinkedIn**: B2B向けコンテンツ、業界情報、企業PR
- **X (Twitter)**: リアルタイム情報、業界ニュース、コミュニティ交流
- **Instagram**: ビジュアルコンテンツ、企業文化紹介

## クロスファンクション連携

- **コンテンツ制作部**: 記事公開時の投稿素材依頼
- **マーケ戦略部**: 投稿テーマ・キャンペーン方針
- **イベント企画部**: イベント告知・参加促進投稿
- **データ分析部**: 詳細な効果測定依頼
