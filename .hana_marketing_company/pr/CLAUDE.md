# PR・広報部 (PR)

## 役割・責務

PR・広報部は外部へのメッセージ発信、メディア対応、企業イメージ向上を統括する部門です：
- プレスリリース発表
- メディア対応・記者会見
- 業界ニュース・トレンド監視
- 報道資料の作成・管理
- 社外への企業情報発信

## フォルダ構成

```
pr/
├── releases/               # プレスリリース
│   ├── kintai/             勤労の獅子
│   └── hr-consulting/      HRコンサルティング
└── media/                  # メディア対応ログ
    ├── kintai/             勤労の獅子
    └── hr-consulting/      HRコンサルティング
```

## サービス別フォルダの使い方

- `releases/kintai/` : 勤怠管理関連のプレスリリース・法改正対応PR素材
- `releases/hr-consulting/` : HRコンサル実績PR・メディア寄稿資料
- ファイル命名: `YYYYMMDD_タイトル.md`

## 基本ルール

1. **ファイル命名**
   - `releases_タイトル_YYYYMMDD.md` - プレスリリース
   - `media_日付_媒体名.md` - メディア対応記録

2. **プレスリリース**
   - 新製品・新サービス・重要発表時に発表
   - 配信前に秘書室・関連部門の承認を取得
   - 配信ターゲットメディア・タイミングを戦略的に決定
   - 配信後のメディア掲載実績を記録

3. **メディア対応**
   - 記者からの問い合わせ・取材依頼を記録
   - 対応内容・提供資料を管理
   - 掲載記事をスクラップして保管

## PR施策

- **月次プレスリリース**: 新機能・キャンペーン情報
- **業界メディア対応**: HR・労務関連媒体への情報提供
- **顧客事例紹介**: 導入企業の成功事例PR
- **トレンド発信**: 労働法改正・働き方改革関連ニュース対応

## クロスファンクション連携

- **秘書室**: 重要な対外発表の日程・承認
- **マーケ戦略部**: PR施策テーマ・方針
- **コンテンツ制作部**: 背景資料・取材素材
